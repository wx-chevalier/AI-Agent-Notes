# 动手学 Agent 开发

目前，Agent 领域整体概念繁杂，缺乏可复用的范式，开发、调试和维护都不容易，尤其是要做真正的企业级应用难度很大。因此，Datawhale 与魔搭社区联合打造了这门《动手学 Agent 应用开发》课程，帮助大家开发的 Agent 不再是“玩具”，而是企业级可用的产品。

当下，国务院发布《深入实施人工智能 + 的行动的意见》，智能体成为衡量智能社会发展的关键指标。但目前 agent 领域概念繁杂、缺乏可复用范式，导致开发者易部署踩坑、不会调试，应用难落地。​

四大模块：提认知、打基础、能实践、拓前沿

- 提认知：由产品专家介绍 Agent 行业状况和市场上的典型产品，帮助大家从产品视角理解企业级 Agent 的开发、运营与维护。
- 打基础：深入理解 Agent、MCP 的原理，从 0 到 1 手动实现最小实践，夯实技术基础。
- 能实践：邀请到来自通义实验室、Dify、Camel-AI 等的专家，讲解成熟的开发范式与最佳实践，分享行业前沿经验。
- 拓前沿：探索快速演进的方向，如端侧 Agent、AI Coding Agent、Model as Agent、智能体通信协议、商业模式等。
