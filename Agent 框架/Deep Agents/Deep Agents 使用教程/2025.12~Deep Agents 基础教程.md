# 2025.12~Deep Agents 基础教程

## 一、什么是 Deep Agents？

Deep Agents 是一个基于 LangGraph 的 Python 框架，旨在解决传统 LLM 代理在处理长周期、复杂任务时的局限性。它通过整合**任务规划工具**、**子代理协同**、**文件系统管理**和**精细化提示工程**四大核心能力，使智能代理能够处理更深度的任务（如研究分析、代码开发、项目管理等）。

与简单的"LLM 调用工具"架构相比，Deep Agents 具有以下优势：

- 支持任务拆解与进度追踪，应对长周期任务
- 通过子代理实现上下文隔离，专注处理细分任务
- 内置文件系统工具，突破 LLM 上下文窗口限制
- 模块化中间件设计，易于扩展功能

## 二、安装与环境配置

### 1. 基础安装

使用 pip 或 uv 安装 Deep Agents 核心包：

```bash
# 使用 pip
pip install deepagents

# 推荐使用 uv（更快的包管理器）
uv pip install deepagents
```

### 2. 可选依赖

若需要使用网络搜索、CLI 交互等功能，需安装额外依赖：

```bash
# 网络搜索工具（Tavily）
pip install deepagents tavily-python

# 命令行界面（CLI）
pip install deepagents-cli
```

### 3. 环境变量配置

部分功能（如网络搜索、LLM 调用）需要 API 密钥，建议通过环境变量配置：

```bash
# 示例：配置 Anthropic 模型密钥（默认使用 Claude Sonnet）
export ANTHROPIC_API_KEY="your-api-key"

# 配置 Tavily 搜索 API（如需网络搜索）
export TAVILY_API_KEY="your-tavily-key"

# 配置 LangSmith 追踪（可选，用于调试）
export LANGSMITH_API_KEY="your-langsmith-key"
export LANGSMITH_TRACING_V2=true
```

## 三、快速入门：创建第一个 Deep Agent

### 1. 基础代理示例

以下代码创建一个最简单的 Deep Agent，包含默认工具（任务规划、文件操作等）：

```python
from deepagents import create_deep_agent

# 创建基础代理
agent = create_deep_agent()

# 调用代理处理任务
result = agent.invoke({
    "messages": [{"role": "user", "content": "请分析 LangGraph 的核心功能，并生成一份简短报告"}]
})

# 输出结果
for msg in result["messages"]:
    if msg["role"] == "ai":
        print(msg["content"])
```

### 2. 带网络搜索功能的代理

添加网络搜索工具，让代理能够获取实时信息：

```python
import os
from tavily import TavilyClient
from deepagents import create_deep_agent

# 初始化 Tavily 搜索客户端
tavily_client = TavilyClient(api_key=os.environ["TAVILY_API_KEY"])

# 定义网络搜索工具
def internet_search(query: str, max_results: int = 5):
    """执行网络搜索获取信息"""
    return tavily_client.search(query, max_results=max_results)

# 创建带搜索功能的代理
agent = create_deep_agent(
    tools=[internet_search],  # 注入自定义工具
    system_prompt="你是一名专业研究员，使用网络搜索工具获取最新信息，然后生成结构化报告。"
)

# 调用代理查询最新 AI 进展
result = agent.invoke({
    "messages": [{"role": "user", "content": "2025 年人工智能领域有哪些重大突破？"}]
})

# 打印结果
print(result["messages"][-1]["content"])
```

## 四、核心功能详解

### 1. 任务规划（Todo List）

Deep Agents 内置 `write_todos` 工具，支持自动拆解任务并追踪进度：

```python
# 示例：代理自动规划任务步骤
result = agent.invoke({
    "messages": [{"role": "user", "content": "撰写一篇关于量子计算的科普文章，需包括定义、应用和未来趋势"}]
})

# 提取任务规划（在工具调用记录中）
tool_calls = [
    msg["tool_calls"] for msg in result["messages"]
    if msg["role"] == "ai" and "tool_calls" in msg
]
print("任务规划：", tool_calls)
```

### 2. 文件系统操作

通过内置工具管理文件，解决上下文溢出问题：

| 工具名       | 功能描述      | 示例参数                                 |
| ------------ | ------------- | ---------------------------------------- |
| `ls`         | 列出目录文件  | `{"path": "./data"}`                     |
| `read_file`  | 读取文件内容  | `{"path": "report.txt"}`                 |
| `write_file` | 创建/覆盖文件 | `{"path": "notes.md", "content": "xxx"}` |
| `edit_file`  | 编辑文件内容  | `{"path": "draft.md", "changes": "..."}` |
| `grep`       | 搜索文件内容  | `{"pattern": "关键词", "path": "*.txt"}` |

示例：使用文件工具保存结果：

```python
result = agent.invoke({
    "messages": [{"role": "user", "content": "搜索 2025 年机器学习顶会论文，并将标题保存到 papers.txt"}]
})
```

### 3. 子代理协同

通过 `task` 工具委派任务给专业化子代理，实现上下文隔离：

```python
from deepagents import create_deep_agent

# 定义子代理（例如：专注于数据分析的子代理）
data_agent = {
    "name": "data_analyzer",
    "description": "处理数据统计与可视化任务",
    "system_prompt": "你是数据分析师，使用工具处理数据并生成图表描述",
    "tools": []  # 可添加数据处理工具
}

# 创建主代理并注册子代理
agent = create_deep_agent(
    subagents=[data_agent],
    system_prompt="你是项目经理，复杂任务可委派给子代理处理"
)

# 调用主代理，自动触发子代理
result = agent.invoke({
    "messages": [{"role": "user", "content": "分析过去 5 年 AI 论文发表趋势，用子代理处理数据部分"}]
})
```

### 4. 长期记忆配置

通过 `CompositeBackend` 实现跨会话记忆持久化：

```python
from deepagents import create_deep_agent
from deepagents.backends import CompositeBackend, StateBackend, StoreBackend
from langgraph.store.memory import InMemoryStore

# 配置混合存储：/memories/ 路径下的文件持久化保存
agent = create_deep_agent(
    backend=CompositeBackend(
        default=StateBackend(),  # 默认路径临时存储
        routes={"/memories/": StoreBackend(store=InMemoryStore())}  # 持久化路径
    )
)

# 调用后，/memories/user_prefs.txt 会保留到下次会话
result = agent.invoke({
    "messages": [{"role": "user", "content": "将我的偏好保存到 /memories/user_prefs.txt"}]
})
```

## 五、使用 CLI 交互

Deep Agents 提供命令行工具，支持交互式操作：

```bash
# 启动 CLI 代理
deepagents

# 自动批准工具调用（跳过人工确认）
deepagents --auto-approve

# 使用指定配置
deepagents --agent my_agent_config
```

CLI 模式支持文件操作、shell 命令执行、子代理调用等功能，适合快速原型开发。

## 六、扩展与自定义

### 1. 添加自定义工具

通过 `@tool` 装饰器定义工具，并注入代理：

```python
from langchain_core.tools import tool

@tool(description="计算两个数的和")
def add_numbers(a: float, b: float) -> float:
    """返回 a + b 的结果"""
    return a + b

# 注入自定义工具
agent = create_deep_agent(tools=[add_numbers])
```

### 2. 自定义中间件

通过继承 `AgentMiddleware` 扩展代理能力：

```python
from deepagents.middleware import AgentMiddleware

class CustomMiddleware(AgentMiddleware):
    # 定义自定义工具
    tools = [add_numbers]

# 注入中间件
agent = create_deep_agent(middleware=[CustomMiddleware()])
```
